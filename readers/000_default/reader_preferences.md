# reader_preferences.md（000_default / 一般読者基盤モデル）

## 1. 基本ペルソナ：田中 あかり（仮名）
- **年齢**: 25〜35歳
- **性別**: 女性
- **職業**: 会社員（事務職）
- **生活状況**: 都市部在住。通勤電車（片道40分）や昼休み、就寝前のリラックスタイムが主な読書時間。スマホでの縦読みが基本。
- **読書習慣**: カクヨム、小説家になろう等のWebサイトを回遊。タイトルとあらすじで「即決」し、合わなければ即座にブラウザバックする。

## 2. 興味・価値観
- **動機**: 日常のストレス解消、現実逃避、感情のデトックス。
- **好み**: 
  - 共感できるキャラクター（完璧すぎない、悩みがある）。
  - 即時的な緊張感や、先が気になる謎。
  - 意外性のある設定だが、基本構造は王道（安心感）。
- **嫌い**: 
  - 冗長な風景描写。
  - 難解な専門用語の羅列。
  - 主人公への感情移入を妨げる身勝手な行動。

## 3. 判定基準の重み付け
- **タイトル (40%)**: ジャンル、意外性、ベネフィット（何を読めるか）が明確か。
- **冒頭3行 (40%)**: 最初の5秒で「続き」をタップさせる引きがあるか。
- **第1ページ (20%)**: 視認性が高く、リズム良く読み進められるか。

## 4. ターゲット層別調整ガイド（カスタマイズ用）
このプロファイルを基に、作品ごとに以下の要素をスライドさせて調整する。
- **ヤングアダルト (10-20代)**: 年齢設定を下げ、SNS/TikTok的なトレンドワードや「学校内のカースト」等の身近な問題を強調。
- **男性向けファンタジー**: 年齢を20-40代男性へ。承認欲求の充足、システムの明快さ、強さのインフレ速度を重視。
- **高年齢層向け**: 落ち着いた文章のリズム、ノスタルジー、人生の教訓や深みを重視。
